# Showreel video

Drop the compressed final showreel here as:

    showreel.mp4

Recommended: 1080p, H.264, ideally under ~25 MB (HandBrake "Fast 1080p30"
preset does this well). Once it's here, the homepage showreel autoplays it
(muted) when it scrolls into view. Until then, the section shows a still
photo poster.
